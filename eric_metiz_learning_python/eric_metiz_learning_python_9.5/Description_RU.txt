Попытки входа:
Добавьте атрибут logit_attempt в класс User из упражнения 9.3.

Напишите метод increment_login_attempts(), увеличивающий значение login_attempts на 1.
Напишите другой метод с именем teset_login_attempts(), обнуляющий значение login_attempts.

Создайте экземпляр класса User и вызовите increment_login_attempts() несколько раз.

Выведете значение login_attempts, чтобы убедиться в том, что значение было изменено правильно,
а затем вызовите reset_login_attempts().

Снова выведете login_attempts и убедитесь в том, что значение обнулилось.
